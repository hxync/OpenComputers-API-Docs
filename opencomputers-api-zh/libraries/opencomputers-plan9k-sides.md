# sides

## 概述

一个支持双向查询的方向表，并附带用于组件方位常量的迭代辅助函数。

## 可用性

- 仓库: `opencomputers`
- 系统: `plan9k`
- 类型: `library`

## 来源

- `OpenComputers/src\main\resources\assets\opencomputers\loot\plan9k\lib\sides.lua`

## 用法

```lua
local sides = require("sides")
```

## 接口

## 备注

- 这个表既能把数字侧面编号映射成名称，也提供 `north`、`south`、`up`、`down`、`forward` 等别名。
