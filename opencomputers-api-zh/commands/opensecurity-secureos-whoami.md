# whoami

## 概述

输出当前 SecureOS 用户名。

## 可用性

- 仓库: `opensecurity`
- 系统: `secureos`
- 类型: `command`

## 来源

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\bin\whoami.lua`

## 语法

```lua
whoami
```

## 用途

把当前的 `USER` 环境变量写到标准输出。

## 示例

```lua
whoami
```
输出当前已登录 SecureOS 用户的用户名。
