# pwd

## 概述

输出当前 SecureOS 工作目录。

## 可用性

- 仓库: `opensecurity`
- 系统: `secureos`
- 类型: `command`

## 来源

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\bin\pwd.lua`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\pwd`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\pwd`

## 语法

```lua
pwd
```

## 用途

把 SecureOS 当前 shell 工作目录写到标准输出。

## 示例

```lua
pwd
```
输出当前 SecureOS 工作目录。
