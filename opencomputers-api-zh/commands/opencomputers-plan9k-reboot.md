# reboot

## 概述

立即重启电脑。

## 可用性

- 仓库: `opencomputers`
- 系统: `plan9k`
- 类型: `command`

## 来源

- `OpenComputers/src\main\resources\assets\opencomputers\loot\plan9k\bin\reboot.lua`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\reboot`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\reboot`

## 语法

```lua
reboot
```

## 用途

立即发出重启请求，让电脑先关机，再立刻重新启动。

## 示例

```lua
reboot
```
立即重新启动电脑。
