# uptime

## 概述

输出电脑已经运行了多久的现实时间。

## 可用性

- 仓库: `opencomputers`
- 系统: `openos`
- 类型: `command`

## 来源

- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\bin\uptime.lua`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\uptime`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\uptime`

## 语法

```lua
uptime
```

## 用途

把机器的运行时长按现实时间的时、分、秒格式输出到标准输出。

## 示例

```lua
uptime
```
输出当前机器的运行时长。
