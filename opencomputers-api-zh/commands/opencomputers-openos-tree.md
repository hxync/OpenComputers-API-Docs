# tree

## 概述

以树状方式显示目录结构，并支持元数据、排序、颜色和深度限制。

## 可用性

- 仓库: `opencomputers`
- 系统: `openos`
- 类型: `command`

## 来源

- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\bin\tree.lua`

## 语法

```lua
tree [OPTIONS] [FILE...]
```

## 用途

遍历一个或多个根路径，并用 Unicode 树枝符号输出每个条目。命令支持长格式元数据、文件大小格式化、深度限制、完整路径输出、引用显示、颜色控制以及可选统计汇总。

## 参数

- `FILE...`: 可选，要打印的根文件或目录。省略时默认使用当前目录。

## 选项

- `-a, --all`: 包含以 `.` 开头的隐藏名称。
- `--full-time`: 与 `-l` 配合时，以完整日期时间形式输出时间戳。
- `-h, --human-readable`: 与 `-l` 配合时，以 1024 进制的人类可读单位输出大小。
- `--si`: 在大小格式化时使用 1000 进制，而不是 1024。
- `--level=LEVEL`: 在每个根路径下最多向下遍历 `LEVEL` 层目录。
- `--color=auto|always|never`: 控制 ANSI 颜色输出。
- `-l`: 显示长格式元数据，包括类型、可写标记、大小和时间。
- `-f`: 为每个条目输出完整路径，而不是仅输出基础名称。
- `-i`: 不显示缩进引导线。
- `-p`: 在目录名称后追加 `/`。
- `-Q, --quote`: 用双引号包裹每个显示出的名称。
- `-r, --reverse`: 在每层目录中反转最终排序结果。
- `-S`: 按文件大小排序。
- `-t`: 按修改时间排序。
- `-X`: 按文件扩展名排序。
- `-C`: 不打印最后的文件和目录数量统计。
- `-R`: 在最终目录/文件统计中把每个根目录本身也计算进去。
- `--help`: 输出内置帮助文本。

## 示例

```lua
tree
```
以树状结构打印当前目录。

```lua
tree --level=2 -a /home
```
打印 `/home`，最多展开两层目录，并包含隐藏条目。

```lua
tree -l -h -p /bin
```
以树状方式显示 `/bin`，并附带长格式元数据、易读大小以及目录后缀 `/`。

## 备注

- 大目录树遍历时，命令大约每两秒会主动让出一次执行机会，以便仍然可以中断。
- 在每层目录中，排序会先按升序完成，再由 `-r` 统一反转。
