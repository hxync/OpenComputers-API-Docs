# date

## 概述

输出当前的游戏内日期和时间。

## 可用性

- 仓库: `opensecurity`
- 系统: `secureos`
- 类型: `command`

## 来源

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\bin\date.lua`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\date`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\date`

## 语法

```lua
date
```

## 用途

使用 OpenComputers 的游戏内时钟，把当前日期和时间输出到标准输出。显示的日期以模拟世界时间线中的 1970 年 1 月 1 日作为纪元起点。

## 示例

```lua
date
```
输出当前的游戏内日期和时间。
