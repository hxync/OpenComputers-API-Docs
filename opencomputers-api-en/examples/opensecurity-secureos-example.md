# example

## Summary

Example SecureOS `rc` service script that demonstrates a simple `start(msg)` hook and persistent in-memory state.

## Availability

- Repository: `opensecurity`
- System: `secureos`
- Type: `example`

## Source

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\etc\rc.d\example.lua`

## Notes

- Its behavior matches the bundled OpenOS example: it prints a welcome explanation, optional configured message, invocation counter, and current runlevel.
