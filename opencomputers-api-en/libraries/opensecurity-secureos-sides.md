# sides

## Summary

Bidirectional side-name table with iteration helpers for component side constants.

## Availability

- Repository: `opensecurity`
- System: `secureos`
- Type: `library`

## Source

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\lib\sides.lua`

## Usage

```lua
local sides = require("sides")
```

## Notes
