# sides

## Summary

Bidirectional side-name table with iteration helpers for component side constants.

## Availability

- Repository: `opencomputers`
- System: `plan9k`
- Type: `library`

## Source

- `OpenComputers/src\main\resources\assets\opencomputers\loot\plan9k\lib\sides.lua`

## Usage

```lua
local sides = require("sides")
```

## Notes

- The table maps numeric side ids to names and also exposes aliases such as `north`, `south`, `up`, `down`, and `forward`.
