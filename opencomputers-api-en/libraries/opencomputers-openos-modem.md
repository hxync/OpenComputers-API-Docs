# modem

## Summary

Software Ethernet-style driver built on OpenComputers modem components and a simple discovery protocol.

## Availability

- Repository: `opencomputers`
- System: `openos`
- Type: `library`

## Source

- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\lib\core\devfs\adapters\modem.lua`

## Usage

local modem = require("network.modem")
