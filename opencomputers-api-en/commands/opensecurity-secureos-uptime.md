# uptime

## Summary

Print how long the computer has been running in real time.

## Availability

- Repository: `opensecurity`
- System: `secureos`
- Type: `command`

## Source

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\bin\uptime.lua`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\uptime`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\uptime`

## Syntax

```lua
uptime
```

## Purpose

Write the machine uptime in real-time hours, minutes, and seconds to standard output.

## Examples

```lua
uptime
```
Print the current machine uptime.
