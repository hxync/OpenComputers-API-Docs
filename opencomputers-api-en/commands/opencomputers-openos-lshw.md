# lshw

## Summary

Display a hardware overview for the current machine.

## Availability

- Repository: `opencomputers`
- System: `openos`
- Type: `command`

## Source

- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\bin\lshw.lua`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\lshw`

## Syntax

```lua
lshw
```

## Purpose

Print device and hardware information detected on the current computer.

## Examples

```lua
lshw
```
Show the current machine's hardware information.
