# reboot

## Summary

Immediately restart the computer.

## Availability

- Repository: `opencomputers`
- System: `plan9k`
- Type: `command`

## Source

- `OpenComputers/src\main\resources\assets\opencomputers\loot\plan9k\bin\reboot.lua`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\reboot`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\reboot`

## Syntax

```lua
reboot
```

## Purpose

Issue an immediate reboot request, shutting the computer down and starting it back up right away.

## Examples

```lua
reboot
```
Restart the computer immediately.
