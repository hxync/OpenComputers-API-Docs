# date

## Summary

Print the current in-game date and time.

## Availability

- Repository: `opensecurity`
- System: `secureos`
- Type: `command`

## Source

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\bin\date.lua`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\date`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\date`

## Syntax

```lua
date
```

## Purpose

Write the current in-game date and time to standard output using OpenComputers' game-time clock. The displayed date is measured from an epoch aligned to January 1, 1970 in the simulated world timeline.

## Examples

```lua
date
```
Print the current in-game date and time.
