# whoami

## Summary

Print the current SecureOS username.

## Availability

- Repository: `opensecurity`
- System: `secureos`
- Type: `command`

## Source

- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\bin\whoami.lua`

## Syntax

```lua
whoami
```

## Purpose

Write the current `USER` environment variable to standard output.

## Examples

```lua
whoami
```
Print the username of the currently logged-in SecureOS user.
