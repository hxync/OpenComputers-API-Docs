# pwd

## Summary

Print the current Plan9k working directory stored in `PWD`.

## Availability

- Repository: `opencomputers`
- System: `plan9k`
- Type: `command`

## Source

- `OpenComputers/src\main\resources\assets\opencomputers\loot\plan9k\bin\pwd.lua`
- `OpenComputers/src\main\resources\assets\opencomputers\loot\openos\usr\man\pwd`
- `OpenSecurity/src\main\resources\assets\opensecurity\lua\SecureOS\usr\man\pwd`

## Syntax

```lua
pwd
```

## Purpose

Write the current value of the `PWD` environment variable to standard output.

## Examples

```lua
pwd
```
Print the current Plan9k working directory.
